# Astro Build

Astro is a modern web framework designed for building fast, content-driven websites. It focuses on delivering optimized performance by shipping **zero JavaScript** by default and supporting multiple UI frameworks like React, Vue, and Svelte.
Using Astro.build to create a simple, fast-loading website that displays all the essential Airbnb guest information when they scan a QR code.

## How to use Astro

To get started with Astro, install it using the following command:

```bash
npm create astro@latest my-astro-site
```

Navigate to your project and start the development server:

```bash
cd my-astro-site
npm run dev
```

Now, you can start building your Astro-powered website by creating pages, components, and layouts.
